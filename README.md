# react-native-iovation

iovation SDK for react native

## Installation

```sh
# using npm
npm install react-native-iovation

# OR using Yarn
yarn add react-native-iovation
```

## Usage

```js
import { getBlackbox } from 'react-native-iovation';

// ...

const blackbox = await getBlackbox();
```
